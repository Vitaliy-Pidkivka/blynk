## Dayry App for Blynk
## `Settings`
- `ctrl + Enter  = add new item`
- `ctrl + click on comment  = remove comment`
- `double click on comment  = set edit mode`
- `lose focus of the comment textarea  = disable edit mode`
- `to set new comment name press 'Enter'`
